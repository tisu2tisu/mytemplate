Pernah denger mysql ?

kalo belum nih baca penjelasannya dulu dari wikipedia :
MySQL adalah sebuah perangkat lunak sistem manajemen basis data SQL (bahasa Inggris: database management system) atau DBMS yang multithread, multi-user, dengan sekitar 6 juta instalasi di seluruh dunia. MySQL AB membuat MySQL tersedia sebagai perangkat lunak gratis dibawah lisensi GNU General Public License (GPL), tetapi mereka juga menjual dibawah lisensi komersial untuk kasus-kasus dimana penggunaannya tidak cocok dengan penggunaan GPL.


Tidak sama dengan proyek-proyek seperti Apache, dimana perangkat lunak dikembangkan oleh komunitas umum, dan hak cipta untuk kode sumber dimiliki oleh penulisnya masing-masing, MySQL dimiliki dan disponsori oleh sebuah perusahaan komersial Swedia MySQL AB, dimana memegang hak cipta hampir atas semua kode sumbernya. Kedua orang Swedia dan satu orang Finlandia yang mendirikan MySQL AB adalah: David Axmark, Allan Larsson, dan Michael "Monty" Widenius.
oke itulah penjelasan dari mysql, sekarang mari kita lihat syntax-syntax nya

Syntax Penjelasan Contoh
Create Database Untuk membuat database baru CREATE DATABASE nama_database;
Create Table Untuk membuat tabel baru CREATE TABLE nama_table;
Alter Table Mengubah struktur tabel ALTER TABLE MODIFY/CHANGE/RENAME/DROP/ENGINE/ADD nama_table;
DROP Untuk Menghapus database/tabel DROP DATABASE/TABLE nama_database/nama_table
DESCRIBE Untuk mendeskripsikan tabel DESCRIBE/DESC nama_tabel;
INSERT Untuk memasukkan data ke dalam table INSERT INTO nama_tabel VALUES ( );
REPLACE Menambah data baru/menambahkan data yang sudah ada REPLACE INTO nama_tabel VALUES ( );
UPDATE untuk mengupdate data UPDATE nama_table SET ...=...;
SELECT untuk melihat isi tabel SELECT * FROM nama_table;
DELETE Untuk menghapus data di dalam table DELETE FROM nama_tabel where nama_kolom=..;
DISTINCT jika ada data yang sama di dalam tabel, maka hanya tampil 1 saja SELECT DISTINCT(nama_kolom) from nama_tabel
